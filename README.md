
# Installation
#### System installtion
```sh
curl -sL https://pbash.pcapis.com/args/install.sh | sudo bash -s -- --system
```

#### User installtion
```sh
curl -sL https://pbash.pcapis.com/args/install.sh | sudo bash -s -- --user
```
